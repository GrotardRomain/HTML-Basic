LES BASES DE L'HTML

- Les balises

- Les balises paires.

Ce qu'est une balise paire:

Une balise paire est une balise qui ne peut être dissociée.

Ex: 

 < p > < / p > 
 < head > < / head> 
 < h1 > < / h1 > 
 < em > < / em > 
 < hr > < / hr >

- Les balises orphelines.

Ce qu'est une balise orpheline:

Une balise orpheline est une balise unique.

Ex: 

 < meta /> 
 < br / > 
 < img / >

Petite soucre ici

- Les attributs

Ce qu'est un attribut:

Un attribut vient compléter les balises pour lui donner des informations supplémentaire. Il se place après le nom de la balise ouvrante. Si on ajoute un attribut dans une balise en paire l'attribut ne s'ajoute que dans la premiere balise.

Ex: 

 < img src = "nomdelaphoto" / >

- La différence entre inline et block

Inline:

Inline, comme son nom l'inquique ces balises permettent de tout garder sur une même ligne:

Ex: Bonjour Bonsoir Bonne nuit

Block

Block, nous pouvons écrire ces balises sur les même ligne mais celle-ci ce mettent automatiquement à la ligne:

Ex:

Bonjour

Bonsoir
Bonne nuit

Ici pour mieux comprendre inline et bloc

- Le markdown et l'HTML

Faire une citation en Mardown versus HTML:

En markdown il suffit de commencer son code par ">" et ensuite d'écrire simplement la citation.

En HTML il y a le 

< blockquote >
pour une citation plus longue avec des paragraphes

Insérer une image en Markdown versus HTML:

En markdown il suffite de taper ![Le nom de votre image](Le lien) et le tour est joué !

En HTML 

 < img = src "votre lien" >

Insérer une vidéo en markdown versus HTML:

En markdown voici le code !::[titre-video](//url-video) .

En HTML 

< video controls src = "video.ogv">Ici la description alternative< / video >
